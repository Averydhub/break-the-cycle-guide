# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**Break the Cycle** is a paid digital healing guide by Avery Hubert (@notAveryHubert, 100k Instagram followers). It targets people recovering from emotionally abusive relationships, trauma bonds, and toxic relationship patterns. Sold directly to her audience.



## Author Voice

Direct, warm, no fluff, no toxic positivity. Conversational but psychologically informed. Short paragraphs. The reader should feel like a knowledgeable friend is talking to them — not a therapist, not a content creator performing wellness. Light profanity is acceptable.

**Hard rule: no dashes anywhere in copy. Ever.**


## Content Structure

7 chapters moving from understanding what happened → detaching → rebuilding → moving forward.

**Converso** (a conversation screenshot analysis tool for detecting red flags) gets one natural mention on the final page only — not pushed aggressively.
